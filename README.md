# tolst-cli
This is the platform for the microfrontends

## How to check that it works?
```
npm start
```

## How to publish?
You can publish new npm version only from the main branch!
```
git add .
git commit -m "-"
npm version patch
// можно посмотреть появился ли тег в git graph
npm run new:version
```
